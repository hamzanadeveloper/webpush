# Demo for the Web Push blog

This is the working demo code for the blog post here:

[https://blog.atulr.com/web-notifications](https://blog.atulr.com/web-notifications)

To launch backend:

```sh
cd backend
npm install
node index.js
```

To launch frontend:

```sh
cd frontend
npm install -g http-server
http-server
```
